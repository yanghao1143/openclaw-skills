# UI/UX 专业设计指南

## 概述

UI/UX Pro Max 是一个全面的设计智能技能，为开发者提供可搜索的专业 UI 风格数据库、无障碍配色方案和优化的字体搭配。支持 Web、移动端和桌面应用的设计。

## 适用场景

- 规划、构建、创建、设计 UI/UX 界面
- 审查和改进现有 UI 代码
- 选择配色方案和字体搭配
- 优化用户体验和交互设计
- 跨平台（Web/iOS/Android/桌面）设计

## 核心资源

### 设计风格库（50+ 种）
- 极简主义、新拟态、玻璃态
- 暗黑模式、赛博朋克、复古
- 企业级、SaaS、电商
- 仪表盘、管理后台

### 配色方案（97 种）
- 按行业分类：科技、医疗、金融、教育
- 按情绪分类：专业、活力、温暖、冷静
- 全部符合 WCAG AA 对比度标准

### 字体搭配（57 种）
- 标题 + 正文组合推荐
- 中文字体搭配建议
- 按风格分类：现代、经典、创意

### UX 准则（99 条）
- 导航和信息架构
- 表单设计和验证
- 加载状态和错误处理
- 移动端触摸交互
- 无障碍访问

## 技术栈支持

React, Vue, Angular, Svelte, Flutter, SwiftUI, Jetpack Compose, Tailwind CSS, CSS Modules
