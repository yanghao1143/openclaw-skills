# Expo 原生 UI 构建

## 概述

Expo 原生 UI 构建技能帮助使用 Expo 和 React Native 创建遵循 iOS 和 Android 平台设计规范的原生 UI 组件。

## 适用场景

- 构建 Expo/React Native 移动应用
- 创建平台特定的 UI 组件
- 实现原生导航模式
- 适配 iOS 和 Android 设计规范
- 优化移动端用户体验

## 核心内容

### 平台适配
- iOS Human Interface Guidelines
- Material Design 3 (Android)
- 平台特定组件选择
- 自适应布局

### 组件开发
- 列表和滚动视图优化
- 手势交互
- 动画和过渡
- 表单和输入控件

### 导航模式
- Stack 导航
- Tab 导航
- Drawer 导航
- Modal 和 Sheet
