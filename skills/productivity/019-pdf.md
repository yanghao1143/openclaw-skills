# PDF 处理工具

## 概述

PDF 处理工具提供全面的 PDF 文档操作能力，包括文本和表格提取、创建新 PDF、合并/拆分文档、处理表单等。当需要程序化处理、生成或分析 PDF 文档时触发。

## 适用场景

- 从 PDF 中提取文本和表格
- 创建新的 PDF 文档
- 合并多个 PDF 文件
- 拆分 PDF 为多个文件
- 填写 PDF 表单
- 批量处理 PDF 文档

## 核心功能

### 文本提取
- 提取全文内容
- 提取表格数据为结构化格式
- 提取图片和元数据
- OCR 支持（扫描件）

### 文档创建
- 从 Markdown 生成 PDF
- 自定义页面布局和样式
- 添加页眉页脚
- 插入图片和表格

### 文档操作
- 合并多个 PDF
- 按页码拆分
- 旋转和裁剪页面
- 添加水印

### 表单处理
- 读取表单字段
- 填写表单数据
- 导出表单数据
