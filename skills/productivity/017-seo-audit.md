# SEO 审计技能

## 概述

SEO 审计技能帮助诊断网站的 SEO 问题，涵盖技术 SEO、页面优化、Meta 标签审查和排名因素分析。当用户提到"SEO 审计"、"为什么排名不好"、"SEO 问题"等关键词时触发。

## 适用场景

- 网站 SEO 健康检查
- 技术 SEO 问题诊断
- 页面 SEO 优化建议
- Meta 标签审查
- 排名因素分析

## 审计清单

### 技术 SEO
- 网站可爬取性（robots.txt, sitemap.xml）
- 页面加载速度
- 移动端适配
- HTTPS 安全
- 结构化数据

### 页面 SEO
- 标题标签优化
- Meta Description
- 标题层级（H1-H6）
- 图片 Alt 文本
- 内部链接结构

### 内容质量
- E-E-A-T 信号
- 内容深度和原创性
- 关键词密度和分布
- 用户意图匹配
